# Log-Out (로그아웃)

> **로그(Log)로 밝혀내는 직장 내 정의, 갑질 빌런 아웃(Out)**
> 머신러닝 텍스트 마이닝과 노동법 RAG 기반 **직장 내 괴롭힘 증거 정량화 · 수사 증강 코파일럿**

제8회 K-디지털 트레이닝 해커톤 · 고용노동부 지정과제

🔗 **소개 페이지:** https://아이디.github.io/logout-demo/ *(배포 후 주소로 수정)*

---

## 한 줄 소개

피해 근로자가 카카오톡 대화 로그(`.txt`)를 업로드하면, 파이썬 엔진이 **발화 독점률 · 업무시간 외 지시 주기성 · 독성 어휘 밀도**를 정량 지표로 연산하고, 고용노동부 매뉴얼·판례만 적재한 **폐쇄형 RAG**로 위반 법 조항을 환각 없이 매칭하여, 근로감독관 제출용 **정량 증거 보고서와 진정서 초안**을 자동 생성합니다.

## 왜 필요한가

- 2024년 직장 내 괴롭힘 신고 **12,253건**(하루 약 50건), 5년 연속 최다
- 피해 경험률 **35.9%** vs 실제 신고 비율 **약 15%** — 격차의 원인은 **입증 역량의 부재**
- 근로감독관은 수만 줄 대화를 밤새 전수조사 → 행정 병목. 2027년까지 감독 물량 **14만 건**으로 급증

→ 피해자의 **입증 공백**과 감독관의 **전수조사 병목**을 하나의 파이프라인으로 해소.

## 처리 흐름

```
카톡 .txt 업로드 (개인정보 자동 마스킹)
      ↓
정량 분석 ML  (발화 독점률 · 시간외 주기성 · 독성 밀도)
      ↓
법리 매칭 RAG  (고용부 매뉴얼 · 판례 Vector DB)
      ↓
생성 · 리포팅  (위험도 스코어링 · 증거보고서 · 진정서 초안)
```

핵심 설계: **근로기준법 제76조의2의 3대 성립요건**을 정량 지표로 매핑
① 지위·관계의 우위 ← 발화 독점률 · ② 업무상 적정범위 초과 ← 시간외 반복 지시 · ③ 신체적·정신적 고통 ← 독성 어휘 밀도

## 3단계 증거 등급

| 등급 | 내용 | 확신 |
|------|------|------|
| **Tier 1** 확정 지표 | 발화 독점률, 시간외 주기성, 반복성 | 수학적 연산 · 반박 불가 |
| **Tier 2** 분류 지표 | 모욕·독성 어휘 밀도 | 모델 확률값 |
| **Tier 3** 정황 후보 | 교묘한 가스라이팅·배제·수동공격 | LLM 맥락 추론 · **최종 판단은 사람** |

## 두 개의 대시보드

- **근로감독관 수사지원 (Main)** — 다건 위험도순 정렬, 핵심 증거 하이라이트, 조항·판례 매칭, 사건 선별
- **피해 근로자 리포트 (Sub)** — 위험도 시각화, 패턴 그래프, 진정서 초안 PDF

## 기술 스택

| 영역 | 사용 기술 |
|------|-----------|
| UI | Streamlit |
| 전처리 | Python · Pandas · Regex |
| 형태소 분석 | KoNLPy |
| 독성 분류 | KcELECTRA · BEEP! · K-MHaS |
| RAG · Vector DB | FAISS · LangChain |
| 생성 LLM | Claude / GPT API |

## 가드레일

- **최종 판단은 사람** — 진위 판별·법적 결정은 근로감독관·노무사·변호사가 수행
- **환각 차단** — 검증된 매뉴얼·판례 출처 범위 내에서만 조항 매칭
- **프라이버시** — 제3자 개인정보 자동 마스킹, 로컬 처리(클라우드 미전송)
